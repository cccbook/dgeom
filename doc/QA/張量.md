# 張量 (Tensor)

* https://gemini.google.com/app/61994261ac313b54

1. 縮並的意義，是將協變和逆變兩者結合，互相取消。
2. 升降指標： 利用基底向量之間的內積關係，透過反變和協變進行轉換，改變視角（座標基底的組合）。
3. 所以張量就是一套代數工具，可以在各種不同坐標系間進行正逆向轉換，透過『張量積，縮並，轉換，升降指標』達到同一個幾何空間上各種運算的工具。
4. 張量的最終目的和價值是：確保物理定律在所有座標系中，都能保持其數學形式的不變性（即一般協變性）。
    * 這裏的物理定律，通常指的是 F(X)-G(Y) = c 這類的，例如 F=ma 寫為 F-ma = 0
    * 也就是在 A 坐標系表達某個方程式，在 B 坐標系該方程式也會成立。
5. 升降指標正是這種轉換。 它是將同一幾何實體從一個張量空間 $V \otimes V$ 映射到另一個張量空間 $V \otimes V^*$ 的操作，因此被視為一種張量類型的轉換或張量操作。
